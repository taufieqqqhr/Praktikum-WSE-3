- Folder "P3 WSE" diorganisir berdasarkan topik (Client-Server, MS, SOA) dan berisi bukti berupa screenshot.

Cara melihat dan menggunakan
----------------------------
1. Lihat laporan: buka file .docx di GitHub viewer atau unduh lalu buka dengan Microsoft Word / LibreOffice / Google Docs.  
2. Lihat screenshot: buka file gambar langsung di GitHub atau unduh dari folder masing-masing topik.

Saran perbaikan (opsional)
--------------------------
- Tambahkan ringkasan singkat pada tiap screenshot (mis. nama eksperimen, apa yang diuji, hasil yang diharapkan/tercapai).  
- Tambahkan README.md di setiap subfolder (Client Server / MS / SOA) untuk konteks detail.  
- Jika ingin menyertakan kode eksperimen, tambahkan file kode (contoh: server, client, Dockerfile) dan instruksi menjalankan (package.json / requirements.txt / run scripts).
